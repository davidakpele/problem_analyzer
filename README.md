## Overview

AI Interactive System is a fully offline, locally running AI desktop application.
It uses llama3 (via Ollama) to understand and respond to your voice or text input.
No data ever leaves your machine.

You speak or type a message. The AI reasons about it and responds in text on screen.
If voice output is enabled, it also reads the response aloud.

---

## Features

- **Voice input** — speak naturally, the system transcribes and processes your speech
- **Camera face detection** — scans your webcam and detects whether a face is present
- **AI reasoning** — all responses generated by llama3 running locally via Ollama
- **Text-to-speech** — AI reads its responses aloud using your system voice engine
- **Voice selector** — choose from all installed voices and adjust speaking speed
- **Problem analysis** — handles math, code bugs, logic, ethics, and general questions
- **Conversation memory** — full history maintained per session
- **No internet required** — completely offline after initial setup

## Installation

### Step 1 — Install Ollama

Download from https://ollama.com/download and install it.

Then pull the llama3 model (one-time, ~4.7 GB download):

```bash
ollama pull llama3
```

### Step 2 — Install Python

Download from https://python.org. On Windows, check "Add Python to PATH" during install.

### Step 3 — Create Virtual Environment

```bash
cd ~/projects/problem_analyzer
python3 -m venv venv
```

Activate it:

```bash
# Linux / macOS
source venv/bin/activate

# Windows (Git Bash)
source venv/Scripts/activate

# Windows (PowerShell)
venv\Scripts\activate
```

You will see `(venv)` at the start of your terminal prompt when active.

### Step 4 — Install Python Packages

```bash
pip install -r requirements.txt
```

Contents of `requirements.txt`:

```
requests
SpeechRecognition
pyaudio
opencv-python
Pillow
pyttsx3
```

### Step 5 — Install TTS Engine

**Windows** — pyttsx3 uses Windows SAPI voices built into the OS. Nothing extra needed.

**Linux** — install espeak-ng:

```bash
sudo apt install espeak-ng
```

### Step 6 — Install Camera and Tkinter (Linux only)

```bash
sudo apt install python3-tk
```

If pyaudio fails to install on Linux:

```bash
sudo apt install portaudio19-dev python3-dev
pip install pyaudio
```

## Running the Application

### Every Time You Run

**Step 1** — Start Ollama in a terminal (keep it open in the background):

```bash
ollama serve
```

**Step 2** — Activate your virtual environment:

```bash
source venv/bin/activate        # Linux
source venv/Scripts/activate    # Windows Git Bash
```

**Step 3** — Run the app:

```bash
python3 ai_system.py
```

### Optional: Launch Script

Create a one-command shortcut on Linux:

```bash
echo '#!/bin/bash
source ~/projects/problem_analyzer/venv/bin/activate
ollama serve &
sleep 2
python3 ~/projects/problem_analyzer/ai_system.py' > run.sh

chmod +x run.sh
./run.sh
```

## How to Use

### Voice Input

Speak naturally into your microphone. No button press needed — the system listens
continuously. Your speech is transcribed and sent to the AI automatically.

> Note: Speech-to-text uses Google Speech Recognition which requires internet.
> Only the transcription step uses internet. All AI reasoning is fully local.

### Text Input

Type your message in the input box at the bottom and press Enter or click SEND.

### Example Conversations

| You say / type                                      | AI does                                        |
|-----------------------------------------------------|------------------------------------------------|
| Hello                                               | Greets you                                     |
| Can you see me?                                     | Checks camera and responds YES or NO           |
| Who am I?                                           | Returns your identity: Admin with full access  |
| What is 144 divided by 12 plus 8?                   | Solves step by step with working shown         |
| Alice and Bob are stranded in the desert...         | Full ethical and probability analysis          |
| My Python code gives an IndexError, how do I fix it?| Identifies bug and suggests a fix              |
| If all humans are mortal and Socrates is human...   | Evaluates the logical argument                 |

### Camera Tips for Best Face Detection

- Ensure your face is well lit from the front
- Avoid strong backlighting such as a window behind you
- Stay within 50-80cm of the camera
- Look directly at the camera lens

## Voice Settings

### Changing the Voice

Click the voice dropdown in the right panel and select any voice from the list.
The change takes effect on the next AI response.

**Windows** — lists all SAPI voices installed on your system (e.g. Microsoft David, Microsoft Zira).

**Linux** — lists espeak-ng voice variants:

```
en          English default
en+m1       English male 1
en+m2       English male 2
en+f1       English female 1
en+f2       English female 2
```

### Adding More Voices on Windows

Go to Settings → Time & Language → Speech → Manage voices → Add voices.
Install any language or voice pack. Restart the app and it will appear in the dropdown.

### Adjusting Speed

Drag the Speed slider (range: 80 to 250 words per minute). Default is 155.

### Testing the Voice

Click TEST VOICE to hear the current voice and speed without sending a message.

## Troubleshooting

### Cannot connect to Ollama

```bash
ollama serve
```

Make sure it shows: `Listening on 127.0.0.1:11434`

---

### "Microphone Required" error at startup

The app requires a microphone to run. Connect one and restart.

---

### No camera detected

The app runs without a camera. Camera features will be disabled.
Connect a USB webcam and restart to enable them.

---

### Voice only speaks once (Linux)

Install espeak-ng — it is more reliable than pyttsx3 on Linux:

```bash
sudo apt install espeak-ng
```

---

### ModuleNotFoundError: tkinter

```bash
sudo apt install python3-tk
```

---

### pyaudio install fails

```bash
sudo apt install portaudio19-dev python3-dev
pip install pyaudio
```

---

### AI response is very slow

llama3 is a large model (4.7 GB). A GPU helps significantly.
You can also use a smaller model by changing `OLLAMA_MODEL` in `ai_system.py`:

```python
OLLAMA_MODEL = "llama3"   # change to "llama3:8b" or "mistral" etc.
```

---

### Speech not recognized

- Check your microphone is set as the default input device
- Speak clearly in a quiet environment
- Check your internet connection (transcription requires it)

---

### No voices in the dropdown

```bash
# Windows
pip install pyttsx3

# Linux
sudo apt install espeak-ng
```

## Project Structure

```
problem_analyzer/
    ai_system.py          Main application (single file)
    requirements.txt      Python package list
    run.sh                Optional launch shortcut (Linux)
    README.md             This file
    venv/                 Virtual environment (auto-created, do not edit)
        bin/              Linux venv binaries
        Scripts/          Windows venv binaries
        lib/              Installed packages
```

### Key Classes in ai_system.py

| Class / Function         | Role                                                        |
|--------------------------|-------------------------------------------------------------|
| `check_mic_available()`  | Checks microphone is connected before launching             |
| `check_camera_available()` | Scans camera indices 0-4 to find a working camera         |
| `build_system_prompt()`  | Builds AI prompt based on available hardware                |
| `query_ollama()`         | Streams responses from the local Ollama API                 |
| `Camera`                 | OpenCV camera loop with dual Haar cascade face detection    |
| `VoiceListener`          | Continuous microphone listener using SpeechRecognition      |
| `Speaker`                | TTS engine — espeak-ng on Linux, pyttsx3 on Windows         |
| `App`                    | Main tkinter GUI — builds UI and wires all components       |
